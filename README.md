# MCPAudit

Poe bot for MCP server security auditing. Paste an MCP server config or code snippet and get an instant security assessment.

Built on [mcp-security-audit](https://github.com/LuciferForge/mcp-security-audit).

## Deploy

```bash
pip install -r requirements.txt
```

Runs on Poe's bot platform via `bot.py`.
